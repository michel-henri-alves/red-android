POLÍTICA DE PRIVACIDADE Última atualização: 27 de fevereiro de 2026 1. Informações Gerais Esta Política de Privacidade descreve como o aplicativo Typpo coleta, utiliza e protege as informações dos usuários. Ao utilizar o aplicativo, você concorda com os termos desta Política. 2. Coleta de Informações O aplicativo pode coletar as seguintes informações: Dados inseridos manualmente pelo usuário (ex: nome de produto, preço, categoria, quantidades). Código de barras capturado pela câmera para identificação de produtos. O aplicativo não coleta dados pessoais sensíveis, tais como: Nome completo CPF ou documento oficial Endereço Dados bancários Localização em tempo real Contatos Fotos armazenadas 3. Uso da Câmera O aplicativo solicita permissão de acesso à câmera exclusivamente para: Leitura de códigos de barras Preenchimento automático de campos relacionados a produtos O aplicativo: Não grava vídeos Não armazena imagens Não envia imagens para servidores Não realiza reconhecimento facial A câmera é utilizada apenas em tempo real para leitura de códigos. 4. Compartilhamento de Dados O aplicativo não vende, aluga ou compartilha dados pessoais com terceiros. Caso haja comunicação com servidor próprio para armazenamento de produtos cadastrados, os dados enviados são limitados às informações necessárias para o funcionamento do aplicativo. 5. Armazenamento e Segurança As informações inseridas no aplicativo podem ser: Armazenadas localmente no dispositivo Enviadas para servidor seguro, quando aplicável Medidas técnicas são adotadas para proteger os dados contra acesso não autorizado. 6. Permissões Utilizadas O aplicativo pode solicitar as seguintes permissões: Câmera (android.permission.CAMERA) Utilizada exclusivamente para leitura de código de barras. Nenhuma permissão é utilizada para fins de rastreamento ou coleta de dados pessoais. 7. Alterações nesta Política Esta Política poderá ser atualizada periodicamente. Recomendamos a revisão regular deste documento. 8. Contato Para dúvidas ou solicitações relacionadas à privacidade, entre em contato: Email: michel.henri.alves@gmail.com
